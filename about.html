About This Website

This website was created for Assignment 4 to improve a GitHub-based website from Assignment 2.

Project Goals

The goal of this website is to show that a GitHub Pages site can be customized with multiple pages, a working navigation menu, and professional styling.

What This Website Includes

  • A live website that can be published through GitHub Pages.
  • Three menu items in the header: Home, Ethics, and About.
  • Separate pages connected through navigation links.
  • A custom CSS file for styling.
  • An IT ethics page with references.
